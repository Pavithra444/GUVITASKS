DIFFERENCE BETWEEN WINDOW AND DOCUMENT OBJECT

  • Both are comes under fundamental components of the javascript document object model
  • Purpose of both the objects are completely different

window

  • This object represents the browser window and provides access to browser related functionalities.
  • It acts as the global object for javascript in a browser environment.
  • Variables and functions those who are declared with global scope will become the properties of the window object.
  • It provides access to various
    1. Properties
    2. Methods
    3. Browser Events

document

  • It represents the HTML content loaded in the browser window
  • The 'document' object represents the document itself.
  • It provides access to elements with in the document.
  • It allows manipulation of the document's
    1. Structure
    2. Content
    3. Style

Properties of window object:

window.closed

window.name

This property sets and returns the name

window.frames

This property returns an array with all window objects in the window.

window.open

This property opens a new browser window.

window.outerHeight

This property returns outerheight of the window.

window.innerHeight

This property returns the innerheight of the window.

window.outerWidth

This property returns outerwidth of the window.

window.innerWidth

This property returns innerwidth of the window.

The above four properties are read only.

Properties of document object:

document.getElementById(id)

In html every element accessed by ducument object with the help of their id.This "getElementById(id)" function returns the particular element by taking it's ID as input.

document.body

It returns the body part of the HTML document.

document.head

Whatever we specified inside the head tag it will be returned.

document.title

It returns the title of the html document.

document.URL

It returns the complete URL of the html document.

document.images

It returns the all images which is referred inside tag in the html document.
